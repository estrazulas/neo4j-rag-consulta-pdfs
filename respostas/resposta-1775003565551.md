**Oque são tensores?**  
No TensorFlow.js um *tensor* é simplesmente **uma estrutura numérica** – ou seja, um conjunto de valores organizados em listas (vetores) ou tabelas (matrizes) que podem ter mais de duas dimensões. Em termos mais práticos, pense em um tensor como **um vetor ou uma lista de números** que o algoritmo de aprendizado de máquina pode ler e manipular. O documento afirma que “o TensorFlow trabalha com tensores, que nada mais são do que vetores ou listas compostas por números”.

**Como são representados em JavaScript?**  
Como o TensorFlow.js roda dentro do ecossistema JavaScript, esses valores numéricos são armazenados **exatamente como arrays** (ou objetos semelhantes a arrays) que contêm apenas números. Não há necessidade de usar objetos complexos ou estruturas de dados especiais; basta transformar os dados em **valores puramente numéricos**.  

Um exemplo prático tirado do contexto é a conversão de informações de pessoas em tensores:

- Cada pessoa tem atributos como **idade**, **cor preferida** e **localização**.  
- A **cor** e a **localização** são categorias discretas, então elas são transformadas em *one‑hot encoding*: se houver três cores diferentes, cada pessoa recebe um vetor onde apenas uma posição contém `1` (a cor da pessoa) e o resto são `0`s.  
- A **idade**, que é um número contínuo, é **normalizada** para o intervalo `[0, 1]` para que não pese mais que os outros atributos.

Depois dessa transformação, tudo fica pronto para ser colocado em um tensor. Em JavaScript isso pode ser visualizado como:

```javascript
// Exemplo ilustrativo (baseado no contexto)
const pessoa = {
  idade: 27,
  corPreferida: "azul",
  localizacao: "São Paulo"
};

// 1) One‑hot para cor (supondo 3 cores: vermelho, verde, azul)
const cores = ["vermelho", "verde", "azul"];
const indiceCor = cores.indexOf(pessoa.corPreferida); // → 2
const oneHotCor = Array(cores.length).fill(0);
oneHotCor[indiceCor] = 1; // → [0, 0, 1]

// 2) Normalizando a idade (exemplo simples: dividir por 100)
const idadeNormalizada = pessoa.idade / 100; // → 0.27

// 3) Montando o tensor (vetor numérico)
const tensorDePessoa = [idadeNormalizada, ...oneHotCor]; 
// tensorDePessoa = [0.27, 0, 0, 1]  (dependendo da ordem escolhida)
```

Nesse trecho vemos que **o tensor resultante é apenas um array de números** (`[0.27, 0, 0, 1]` no exemplo). Cada posição do array tem um significado definido pela forma como os dados foram codificados (idade normalizada + os “one‑hot” das categorias).

**Resumo rápido**

- **Tensor = lista/vetor de números** que o TensorFlow.js pode processar.  - Em JavaScript, esses números são armazenados em **arrays** ou objetos semelhantes.  
- Para usar dados do mundo real (como atributos de pessoas), primeiro **converte‑os em valores numéricos** (normalização, one‑hot encoding, etc.) e então **empacota‑os em um array** que será o tensor.  

Essa representação numérica é a ponte que permite que algoritmos de aprendizado de máquina “entendam” dados originalmente expressos em estruturas de JavaScript mais naturais, como objetos ou listas. Assim, ao transformar objetos em tensores, você está preparando-os para serem manipulados por redes neurais e, finalmente, para fazer previsões ou classificações.
